**Worker Recuperation** refers to
<a href="Working_Class" class="wikilink" title="workers">workers</a>
taking over their former workplaces and running them as
<a href="Worker_Cooperative" class="wikilink"
title="worker cooperatives">worker cooperatives</a> within a
<a href="Capitalism" class="wikilink" title="capitalist">capitalist</a>
economy. This usually happens after a workplace goes bankrupt, as
opposed to a
<a href="Work-In" class="wikilink" title="work-in">work-in</a> which
employs <a href="Workers&#39;_Self-Management" class="wikilink"
title="temporary self-management">temporary self-management</a> as a
strike tactic.

## Examples

- 1975: Whyalla Glove Factory (Australia)
- 1981: Pantelemidis Factory (Greece)
- 1983: Koulistanidis Textile Factory (Greece)
- 1985: Pascual Boing (Mexico)
- 2001: <a href="Argentinazo" class="wikilink"
  title="Argentinazo">Argentinazo</a>
  - FaSinPat
  - Hotel Bauen
  - Brukman Factory
  - Commerce and Justice
- 2003: Flasko (Brazil)
- 2006: <a href="Jugoremedija" class="wikilink"
  title="Jugoremedija">Jugoremedija</a> (Serbia)
- 2007:
  <a href="PT_Istana" class="wikilink" title="PT Istana Takeover">PT
  Istana Takeover</a> (Indonesia)
- 2012:
  <a href="New_Era_Windows" class="wikilink" title="New Era Windows">New
  Era Windows</a> (USA)
- 2012: Ri-Maflow (Italy)
- 2012: Kilkis Hospital Takeover (Greece)
- 2013: Viome Takeover (Greece)
- 2013: ERT radio station takeover (Greece)
- 2014: Kazova Textile Factory takeover (Turkey)
- 2015: <a href="Dita_(Factory)" class="wikilink"
  title="Dita Detergent Factory Takeover">Dita Detergent Factory
  Takeover</a> (Bosnia and Herzegovina)
