## Operating Systems

### <img src="Gnulinux.svg" title="Gnulinux.svg" width="64"
alt="Gnulinux.svg" /> General

- It's not just a Windows and Mac world. **Mac vs. PC is a marketing
  illusion**, and [Apple doesn't respect our
  freedom](http://www.defectivebydesign.org/apple).
- Many of the programs listed below are developed on
  <a href="Wikipedia:GNU_Linux.md" class="wikilink"
  title="GNU/Linux">GNU/Linux</a>, and then
  <a href="Wikipedia:Porting.md" class="wikilink" title="ported">ported</a>
  to Windows. As such, they're likely to work better on GNU/Linux. See
  the <a href="Free_Skool:Software#GNU.2FLinux.md" class="wikilink"
  title="GNU/Linux options">GNU/Linux options</a> below if you'd like to
  try out a new and exciting operating system, and get out of the Mac
  vs. PC rut.
- Due to a malfunction in
  <a href="User:Diggity.md" class="wikilink" title="Diggity">Diggity</a>'s
  reality distortion field, most of the individual apps listed below are
  for Windows. That [awful proprietary OS](http://en.windows7sins.org)
  is still widely installed, so it can be a starting point for FOSS
  users.
- Suggestions for Mac OSX programs are welcome, and this list will also
  grow to include phone/tablet platforms.
- Suggestions for software without Windows or Mac builds should have an
  accompanying note. GNU/Linux users generally don't need to search
  around the Web and download individual packages/tarballs/binaries,
  since modern distributions have
  <a href="Wikipedia:Package_management_system.md" class="wikilink"
  title="package management systems">package management systems</a>.

### <img src="Usb-creator.svg" title="Usb-creator.svg" width="48"
alt="Usb-creator.svg" /> "Live" DVD/USB

- All of the operating systems listed above have a "try before you
  install" option, where you can [run them as a
  LiveDVD/USB](https://en.wikipedia.org/wiki/Live_cd) and not touch your
  local files or hard drives.
- [HowTo Burn an ISO to a Live
  DVD/USB](https://help.ubuntu.com/community/BurningIsoHowto)
- [How try a LiveCD ISO in
  VirtualBox](https://www.youtube.com/watch?v=pcw25WleOs4)

### <img src="Wine-uninstaller.svg" title="Wine-uninstaller.svg" width="48"
alt="Wine-uninstaller.svg" /> Running Windows Apps in GNU/Linux

- [Wine](https://en.wikipedia.org/wiki/Wine_%28software%29): Software
  compatibility layer for GNU/Linux that will run Windows software
  natively. Check the [AppDB](http://appdb.winehq.org) to see if your
  program will run on Wine, and how well.
  - [A simple Wine
    tutorial](http://www.howtogeek.com/105271/how-to-run-windows-software-on-ubuntu-with-wine/)
- [PlayOnLinux](http://www.playonlinux.com) and
  [Crossover](http://www.codeweavers.com/compatibility/): Two options
  built upon Wine. Use Crossover's [Compatibility
  DB](http://www.codeweavers.com/compatibility) to search for the
  program you're trying to run.
- [VirtualBox](https://en.wikipedia.org/wiki/Virtualbox): Run Windows
  inside a [virtual
  machine](https://en.wikipedia.org/wiki/Virtual_machine). Also a great
  way to try out [GNU/Linux and Unix distros](http://distrowatch.org)!

### <img src="Tor_logo1.png" title="Tor_logo1.png" width="48"
alt="Tor_logo1.png" /> Privacy & Anonymity

- [TAILS](https://tails.boum.org): The Amnesiac Incognito Live System is
  a live GNU/Linux distribution. It will boot without leaving any traces
  on your hard drives, like any other GNU/Linux LiveDVD/USB. However, it
  also sends all network traffic through
  [Tor](https://www.torproject.org/) for anonymity and has a suite of
  encryption tools pre-installed.
- [Whonix](http://whonix.sf.net/): Designed to run in [virtual
  machines](https://en.wikipedia.org/wiki/Virtual_machine), using
  software such as
  [VirtualBox](https://en.wikipedia.org/wiki/Virtualbox). Read [more
  about this approach](https://lwn.net/Articles/520106/) before trying.
- See our section on
  <a href="Libre_Recommendations#Web_Browsers.md" class="wikilink"
  title="Web Browsers">Web Browsers</a> for private browsing options.

## Graphics

### Libre Graphics Magazine

- [A great resource for artists](http://libregraphicsmag.com/), produced
  collaboratively under FOSS/free culture licenses. All issues are
  available in low and high res on a "payment optional" model.
  - [Libre Graphics Asset page](http://libregraphicsmag.com/asset/),
    [Manifesto](http://libregraphicsmag.com/manifesto/), and [source

### Photo Editors

- [GIMP](http://gimp.org): Edits photos and creates [raster
  graphics](https://en.wikipedia.org/wiki/Raster_graphics), as well as
  plugins for animation. Replaces Adobe Photoshop, Microsoft Photo
  Editor.
- [Paint.NET](http://getpaint.net): A popular alternative. Requires an
  updated version of the .NET framework (i.e. an up-to-date version of
  Windows).
- [Pinta](http://pinta-project.com/): Simple multi-platform photo
  editor. Like Paint.NET but with a more basic interface.
- [Xara Xtreme](http://www.xara.com/us/products/designer/): Photo and
  graphic editor.
- [LightZone](http://lightzoneproject.org): Manages photo collections
  and does basic editing, supports RAW.
- [Raw Therapee](http://www.rawtherapee.com/): Advanced photo collection
  management.
- [darktable](http://www.darktable.org/): Photography workflow
  management and RAW editor.

### Drawing

- [Inkscape](http://inkscape.org): Creates
  [SVG](https://en.wikipedia.org/wiki/Svg) [vector
  graphics](https://en.wikipedia.org/wiki/Vector_graphics), exports to a
  variety of vector and [raster
  graphics](https://en.wikipedia.org/wiki/Raster_graphics) formats.
  Replaces Adobe Illustrator.

### Animation

- [Synfig](http://www.synfig.org/cms/): 2-D animation studio. Replaces
  Adobe Flash.

### Fonts

- [Open Font Library](http://www.openfontlibrary.org/)
- [OSP Foundry](http://ospublish.constantvzw.org/foundry/)

### Misc

- [MWsnap](http://www.mirekw.com/winFreeware/mwsnap.html): Simple
  screenshot utility.
- [Colorpic](http://www.iconico.com/colorpic/): Color picker (great for
  figuring out [hex color
  codes](https://en.wikipedia.org/wiki/Web_colors#Shorthand_hexadecimal_form)
  in HTML/CSS!)
- [Build-Your-Own Adobe Creative Suite using libre and/or gratis
  alternatives](http://lifehacker.com/5976725/build-your-own-adobe-creative-suite-with-free-and-cheap-software)

#### Built for Privacy & Anonymity

- [Tor Browser
  Bundle](https://www.torproject.org/projects/torbrowser.html.en):
  Anonymous Web browsing. Your communications are bounced around the
  [Tor anonymity
  network](https://en.wikipedia.org/wiki/Tor_%28anonymity_network%29).
- [Pirate Browser](http://piratebrowser.com/): A browser by
  [TPB](http://en.wikipedia.org/wiki/The_Pirate_Bay) that bundles Tor,
  Firefox Portable, Foxy Proxy, and some custom configs that allows you
  to circumvent censorship that certain countries impose on their
  citizens (e.g. Iran, North Korea, United Kingdom, The Netherlands,
  Belgium, Finland, Denmark, Italy, Ireland).
  - **Note:** It's unclear [whether or
    not](http://www.theregister.co.uk/2013/08/12/pirate_bay_releases_new_privacyminded_browser/)
    it gives you all the anonymity of the [Tor Browser
    Bundle](https://www.torproject.org/projects/torbrowser.html.en).

#### Browser Tips

- **Set up your browser to be smart and secure by [following these
  tips](http://fixtracking.com/), no matter what browser you're using!**
- Escape the [Filter Bubble](http://dontbubble.us/) and [avoid online
  tracking](http://donttrack.us/).
- Utilize [DuckDuckGo](https://duckduckgo.com),
  [OpenStreetMap](http://www.openstreetmap.org/),
  [Startpage](https://www.startpage.com/),
  [Ixquick](https://www.ixquick.com/), and
  <a href="Software_Recommendations#Riseup.net.md" class="wikilink"
  title="Riseup.net">Riseup.net</a>'s services instead of relying upon
  Google (and its much smaller "rivals").
  - Please read our
    <a href="DuckDuckGo.md" class="wikilink" title="page on DuckDuckGo">page on
    DuckDuckGo</a> for an overview of potential problems with the
    alternative search engines, including DuckDuckGo. The search engine
    waters are very murky, but the alternatives are still much better
    than Google, Bing, Yahoo!, et al.
- Check out [RiseUp.net](https://riseup.net)'s [Browser Privacy
  Scorecard](https://help.riseup.net/en/browser-privacy-scorecard) to
  rate your browser.
- **Don't use CryptoCat!** CryptoCat, Hushmail, and other tools that
  utilize [host-based
  security](http://www.schneier.com/blog/archives/2012/08/cryptocat.html)
  are full of false promises. Also, [DeCryptoCat has
  proven](http://tobtu.com/decryptocat.php) the tool can't be trusted.

### E-mail & Calendar

- [Mozilla Thunderbird](https://www.mozilla.org/thunderbird): E-mail
  client and [RSS feed](https://en.wikipedia.org/wiki/Rss_feed) reader.
  Replaces Microsoft Outlook.
  - plugins: [Lightning
    Calendar](https://addons.mozilla.org/en-US/thunderbird/addon/lightning/),
    [EnigMail](https://addons.mozilla.org/en-us/thunderbird/addon/enigmail/),
    [Adblock
    Plus](https://addons.mozilla.org/en-US/thunderbird/addon/adblock-plus/)
- [Sunbird](https://www.mozilla.org/projects/calendar/sunbird):
  Stand-alone calendar program. Can also be combined with [Mozilla
  Thunderbird](https://www.mozilla.org/thunderbird) as [Lightning
  Calendar](https://addons.mozilla.org/en-US/thunderbird/addon/lightning/).
- [Bitmessage](https://bitmessage.org): Private messaging software based
  on <a href="Bitcoin.md" class="wikilink" title="Bitcoin">Bitcoin</a>,
  intended for e-mail-like communication but with [much more
  potential](https://bitcointalk.org/index.php?topic=212490.0).

#### Hosting

- Private e-mail and calendar requires a privacy-respecting host.
  Lavabit and Silent Mail have closed down, and there aren't many
  options for third-party providers left. Consider the
  <a href="Libre_Recommendations#Riseup.net.md" class="wikilink"
  title="Riseup.net">Riseup.net</a> option (although you have to be
  invited to get an @riseup.net account) or paying for a service like
  [MyKolab](http://mykolab.com) (it's recommended by [pj at
  groklaw](http://www.groklaw.net/article.php?story=20130818120421175)).
  If you're going to host your own e-mail, there are a lot of options
  and you may want to [install Kolab yourself](http://www.kolab.org/).

### IM & Chat

- [Pidgin](http://pidgin.im): Instant Messaging client for (almost) all
  chat protocols, including IRC. Replaces AIM, Google Talk, Facebook
  Messenger, ICQ.
  - plugins: [Off-the-Record messaging](http://www.cypherpunks.ca/otr/),
    [Guifications](http://sourceforge.net/projects/guifications/)
- [SIP](https://en.wikipedia.org/wiki/Session_Initiation_Protocol)
  Video/Voice Calls: [Jitsi](https://jitsi.org/) and
  [Ekiga](http://www.ekiga.org/) are two choices to replace Skype and
  other VoIP clients.
- [XChat](http://xchat.org/): Allows you to join multiple IRC channels,
  chat one-on-one, and transfer files.
- [Bitmessage](https://bitmessage.org): Private messaging software based
  on <a href="Bitcoin.md" class="wikilink" title="Bitcoin">Bitcoin</a>,
  intended for e-mail-like communication but with [much more
  potential](https://bitcointalk.org/index.php?topic=212490.0).
- [BitTorrent
  Chat](http://labs.bittorrent.com/experiments/bittorrent-chat.html):
  Chat privately using decentralized hash tables and keys over
  Bittorrent.
- [Tox](http://tox.im/): A secure messaging replacement for Skype, still
  in testing.
- [QuickMSG](http://quickmsg.vreeken.net/): A text message app for
  Android that uses PGP-encrypted e-mail with IMAP/SMTP instead of SMS.
  It's like sending an e-mail with
  [Enigmail](https://enigmail.net/home/index.php), but it looks like SMS
  instead.

### File Transfer & Sharing

- [Filezilla](http://filezilla-project.org/):
  [FTP](https://en.wikipedia.org/wiki/FTP) and [SFTP
  client](https://en.wikipedia.org/wiki/SSH_file_transfer_protocol) for
  transferring files. Replaces CuteFTP.
- [Deluge Torrent](http://deluge-torrent.org): Downloads via the
  [Bittorrent](https://en.wikipedia.org/wiki/BitTorrent) protocol, for
  efficient downloading of large binary files. Perhaps best known from
  sites such as [PirateBay](https://en.wikipedia.org/wiki/Piratebay)
  (NSFW), but also may be used with [legal
  torrents](http://gigaom.com/2007/03/03/ten-sites-for-free-and-legal-torrents/).
  Deluge includes advanced [encryption
  settings](https://en.wikipedia.org/wiki/Deluge_%28software%29#Features).
  - Instructions for
    <a href="Bittorrent.md" class="wikilink" title="setting up Deluge">setting
    up Deluge</a>.
- [RetroShare](http://retroshare.sourceforge.net/):
  [Peer-to-peer](https://en.wikipedia.org/wiki/Peer-to-peer)
  ("friend-2-friend"), and secure, decentralised sharing. Replaces
  Limewire.
  - [Screenshots and instructions for set
    up](http://retroshare.sourceforge.net/screenshots.html)
- [SparkleShare](http://sparkleshare.org/): Simple file synchronization
  and [cloud
  storage](https://en.wikipedia.org/wiki/Cloud_storage_service), using
  [git](https://en.wikipedia.org/wiki/Git_(software)) as a backend.
  Replacement for Dropbox, [Ubuntu One](https://one.ubuntu.com/).
- [OneSwarm](http://www.oneswarm.org/): A file sharing program that
  utilizes
  <a href="Bittorrent.md" class="wikilink" title="Bittorrent">Bittorrent</a>
  for strong anonymity.
  - Still not sure what
    <a href="Bittorrent.md" class="wikilink" title="Bittorrent">Bittorrent</a>
    is? Familiarize yourself with it through this great paper, [Why My
    Printer Received a DMCA Takedown
    Notice](http://www.michaelpiatek.com//papers/piatek_hotsec08.pdf).

#### Bad Peers

- [PeerGuardian](http://sourceforge.net/projects/peerguardian/) and
  [PeerBlock](http://www.peerblock.com/) are firewalls to block known
  bad addresses from investigating you.
  - [More info
    here](http://netforbeginners.about.com/od/downloadingfiles/a/peerblock_p2p_anonymity.htm)
- [MoBlock](https://help.ubuntu.com/community/MoBlock) and
  [PGL](http://sourceforge.net/p/peerguardian/wiki/pgl-Main/) are
  alternatives for GNU/Linux.
