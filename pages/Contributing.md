This is a wiki by Anarchists for those interested in Anarchism.

As such articles will unsurprisingly take the basic position that
freedom from hierarchy is good, and that forms of hierarchy (state,
social, capital etc.) are negative.

Anarchist contributors are encouraged to ensure accuracy in their
contributions, to not use this wiki for the promotion of any particular
form of anarchism, and to account for differing opinions.

Non-Anarchists can edit and create new articles, but only if done in
good faith and if they meet the purpose, expectations and standards of
this wiki.

## Templates

Templates are a quick way to pre-populate a new wiki article with the
needed headings, sections, categories and other elements

### Using Templates

- Add the template call:
      {{subst:NAME}}
- Save the page (not just preview)
- Edit the page again - you'll now see the template content has been
  inserted as regular wikitext that you can fill in

Some current templates you may wish to use are:

- "<a   href="Template:Historical.md" class="wikilink"
  title="Historical">Historical</a> - For events and periods of time
- "<a   href="Template:Geographical.md" class="wikilink"
  title="Geographical">Geographical</a> - For places or regions
- "<a   href="Template:Movements.md" class="wikilink"
  title="Movements">Movements</a> - For groups, societies and
  organisations
- "<a   href="Template:Theoretical.md" class="wikilink"
  title="Theoretical">Theoretical</a> - For explaining Anarchist theory
  or critiquing other concepts
- "<a   href="Template:Book-Summary.md" class="wikilink"
  title="Book-Summary">Book-Summary</a> - For summarising or critiquing
  theory and fiction books
- "<a   href="Template:Biography.md" class="wikilink"
  title="Biography">Biography</a> - For people (humans and other
  sentient beings)

Each template contains suggested categories - delete those not
appropriate to the article, and add any existing ones as needed - a
"<a   href="Special:Categories.md" class="wikilink"
title="list of those available is here">list of those available is
here</a> (Add new categories cautiously)

### Creating New Templates

Create a page with the name - Template:NAME

Most templates should have -

- A title heading
- A references heading and section
- An inline image
- An infobox (coming soon)

### InfoBoxes

Infoboxes can be created here -
"<a   href="Special:InfoboxBuilder.md" class="wikilink"
title="Special:InfoboxBuilder">Special:InfoboxBuilder</a>

Or manually like this - "<a   href="Template:InfoBox-Test.md" class="wikilink"
title="Template:InfoBox-Test">Template:InfoBox-Test</a>

To include an InfoBox in an article -

     {{BoxNAME
     |image = Example.jpg
     |caption = Image caption
     |field1 = Value1
     }}

Different InfoBox templates have different values. See the Templates
above for other examples.

Infoboxes should appear here (after some delay) -
"<a   href="Special:AllInfoboxes.md" class="wikilink"
title="Special:AllInfoboxes">Special:AllInfoboxes</a>

Here is a tutorial on using them -
<https://portability.fandom.com/wiki/Portable_Infoboxes>

More Infobox Tags -
<https://community.fandom.com/wiki/Help:Infoboxes/Tags>
